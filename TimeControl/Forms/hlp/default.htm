Учет рабочего времени - приложение, предназначенное для просмотра информации по проходам сотрудников на рабочее место.

Сотрудник компании видит данные по проходам сотрудников на рабочее место:

  • обобщенные данные по сотрудникам, которые подчинены непосредственно этому сотруднику, так же сотрудник имеет возможность посмотра детализированной информации по проходам подчиненных сотрудников.
  • обобщенные данные по проходам самого сотрудника. Сотрудник не имеет права просмотра информации других сотрудников, если они не подчинены ему.
  • обобщенные и детализированные данные по всем сотрудникам, если сотрудник является кадровиком/охранником.
Основы работы с приложением:

Поля фильтрации:
  • Компания - выбранные компании, возможно 4 варианта: 1. Элементы из списка компаний, 2. Элементы за исключением списка компаний, 3. Любая компания, 4. Компания не указана.
  • Выбранные компании влияют на значения других фильтров: 1. Подразделение, 2. Должность, 3. Сотрудник.
  • Подразделение - выбранные подразделения, возможно 4 варианта: 1. Элементы из списка подразделений, 2. Элементы за исключением списка подразделений, 3. Любое подразделение, 4. Подразделение не указано.
  • Выбранные подразделения влияют на значения других фильтров: 1. Должность, 2. Сотрудник.
  • Должность сотрудника - выбранные должности, возможно 4 варианта: 1. Элементы из списка должностей, 2. Элементы за исключением списка должностей, 3. Любая должность, 4. Должность не указана.
  • Выбранные должности влияют на значение фильтра: Сотрудник.
  • Сотрудник - выбранные сотрудники, возможно 4 варианта: 1. Элементы из списка сотрудников, 2. Элементы за исключением списка сотрудников, 3. Любой сотрудник, 4. Сотрудник не указан.
  • Период - выбор типа периода: 1. день, 2. неделя, 3. месяц, 4. квартал, 5. год, 6. произвольный период.
  • При выборе периода (1-5) доступны элементы управления, листающие выбранный период.
  • Возможен выбор произвольного периода, путем установки выбранных дат в календарях.
  • Нарушения пропускного режима - определяет каким образом будут учитываться нарушения пропускного режима. 1. В пользу сотрудника, 2. В пользу компании.
  • Отображать сотрудников, имеющих проходы за указанный период - при выборе этой опции применяются все фильтры, включая выбранный период.
  • В противном случае применяются фильтры, за исключением выбранного периода. Также в список включаются сотрудники имеющие должность и/или карточку для прохода.
Для того, чтобы посмотреть данные проходов сотрудников, удовлетворяющие условиям фильтрации нажмите на Искать

Управляюшие элементы:
  • Очистить - при нажатии очищает установленные критерии фильтрации и таблицу с данными проходов сотрудников.
  • Искать - при нажатии осуществляется поиск и вывод данных проходов сотрудников, удовлетворяющих условиям фильтрации.

Таблица с данными проходов сотрудников:
  • Пиктограмма детализации проходов, доступна при наличии проходов у сотрудника. В зависимости от выбранного периода клик на пиктограмме приводит к показу:
  • 1. При выбранном периоде: день - отображается детализация проходов за этот день.
  • 2. При любом другом выбранном периоде - отображаются проходы выбранного сотрудника за этот период.
  • Сотрудник - ФИО сотрудника, с возможностью сортировки.
  • Суммарное время работы - время работы сотрудника за выбранный период, с возможностью сортировки, с указанием нарушений пропускного режима (если есть нарушения, время помечается *).
