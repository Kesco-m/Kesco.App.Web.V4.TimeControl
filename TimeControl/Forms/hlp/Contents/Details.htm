Детализированный отчёт по проходам сотрудника за выбранный период

Данная форма предназначена для просмотра детализированной информации по проходам сотрудника за выбранный период.

Поля фильтрации:
  • Период - выбор типа периода: 1. день, 2. неделя, 3. месяц, 4. квартал, 5. год, 6. произвольный период.
  • При выборе периода (1-5) доступны элементы управления, листающие выбранный период.
  • Возможен выбор произвольного периода, путем установки выбранных дат в календарях.
  • Не показывать пустые записи - при выборе этой опции скрываются дни, в которые у сотрудника не было проходов.
  • Нарушения пропускного режима - определяет каким образом будут учитываться нарушения пропускного режима. 1. В пользу сотрудника, 2. В пользу компании.
Для того, чтобы посмотреть данные проходов сотрудников, удовлетворяющие условиям фильтрации нажмите на Искать

Управляюшие элементы:
  • Список сотрудников - при нажатии перенаправляет на страницу с данными проходов сотрудников.
  • Искать - при нажатии осуществляется поиск и вывод данных проходов сотрудника, удовлетворяющих условиям фильтрации.

Таблица с данными проходов сотрудника:
  • Пиктограмма детализации проходов, доступна при наличии проходов у сотрудника. Клик на пиктограмме приводит к показу детализация проходов за этот день.
  • Дата - день из выбранного периода, с возможностью сортировки.
  • начало работы - время начала работы, первого прохода в офис за этот день, с возможностью сортировки.
  • Конец работы - время окончания работы, последнего выхода из офиса за этот день, с возможностью сортировки.
  • В случае перерыва в работе, превышающего 8 часов, отображаются периоды присутствия на рабочем месте в этот день.
  • Суммарное время работы - время работы сотрудника за день, с возможностью сортировки, с указанием нарушений пропускного режима (если есть нарушения, время помечается *).
  • Суммарное время перерывов - время перерывов сотрудника за день, с возможностью сортировки.
